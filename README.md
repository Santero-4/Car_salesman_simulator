# Car_salesman_simulator
A text-based game about buying and selling used cars

to run:
Either open in your code editor of choice and run `main.java` or run it from the command line with `java main.java`

You can type `help` at any time during the game to see the tutorial again, if you need even more hints there's a `cheatsheet.md` in the game's files that you can check out
